# Changelog

All notable changes to this project will be documented in this file.

## 1.0.0 - 2025-08-27
- Major: First stable release of TSN test case generator
- Features: CLI commands (generate, validate, lint, info), domain services (topology, stream, route generation)
- Docs: Architecture diagrams, file format specs

## 0.1.0 - Initial release
- Poetry packaging
- Typer CLI with generate, validate, info
- pipx installable
- Man page and shell completion

## Versioning Policy
- This project follows [Semantic Versioning (SemVer)](https://semver.org/): MAJOR.MINOR.PATCH
- Breaking changes increment MAJOR, new features increment MINOR, bugfixes increment PATCH
